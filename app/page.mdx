# Maitask Documentation

Maitask is a modular automation platform with three core product surfaces:

- **Plane Frontend**: operator console for package management, execution control, workflows, schedules, notifications, and billing.
- **Plane**: control-plane API for identity, package registry, credits, workflows, scheduling, notifications, and runtime orchestration.
- **Runtime**: high-performance execution engine for package loading, sandboxed execution, adapters, and service-to-service integration with Plane.

## Documentation Scope

This site is the canonical platform documentation for:

- Product architecture and interaction model
- Service bootstrapping and configuration
- Runtime and Plane API contracts
- Security, service authentication, and operational runbooks
- SDK and integration references

## Source of Truth

- Plane codebase: `plane/src`
- Runtime codebase: `runtime/src`
- Plane Frontend codebase: `plane-frontend/src`

The pages in this site are maintained to match those implementations.
