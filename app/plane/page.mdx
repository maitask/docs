# Plane API

Plane is the control-plane service for identity, package registry, execution orchestration, scheduling, workflow orchestration, credits, payments, notifications, and analytics.

## Service Responsibilities

- Authentication and authorization
- Package publish/share/visibility management
- Execution record persistence and status updates
- Runtime gateway and orchestration
- Credits and payment flows
- Workflow and schedule orchestration
- Notification management and user preferences
- Metrics, analytics, and audit logging

- `/analytics`

- `/roles`
- `/organizations`
- `/ws`

OpenAPI JSON is exposed at `/api-docs/openapi.json` and Swagger UI at `/swagger-ui`.

## Runtime Integration Notes

- Plane runtime integration supports HTTP and gRPC.
- In hybrid mode, supported operations use gRPC; selected operations intentionally fallback to HTTP.
- Runtime load-balancing strategy is configurable (`round_robin`, `least_connections`, `random`, `health_aware`).
